Gooner Dev

Arsenal fanclub and android developer

GoonerDev เกิดจากคำสองคำมารวมกัน นั่นคือ Gooner และ Dev โดย Gooner เป็นชื่อเรียกกลุ่มบุคคลที่เป็นแฟนตัวยงของทีมฟุตลอล ปืนใหญ่ อาร์เซนอล ในประเทศอังกฤษ และคำว่า Dev มาจากคำว่า Developer ซึ่งแปลว่า นักพัฒนา เหตุก็เนื่องจากว่า ตัวเราเป็นแฟนอาร์เซนอล และก็เป็น android developer ด้วย จึงใช้ชื่อนี้เป็นชื่อบล็อก

GoonerDev เป็นบล็อกที่เราเขียนไว้ โดยหวังว่าจะเป็นประโยชน์กับสหายที่มาอ่านทุกๆท่านไม่มากก็น้อย หากมีสิ่งใดผิดพลาด หรืออยากแนะนำ ก็เชิญได้ที่ facebooke ของเราได้

ขอให้สหายทุกท่าน เมื่ออ่านแล้ว ก็จงคิดวิเคราะห์และลองทำด้วยตัวเองก่อน เผื่อว่าจะผิดถูกอย่างใด ก็จะได้ช่วยแนะนำกัน ถือเป็นการแลกเปลี่ยนความรู้ครับ
