Font size บน android ทำไมต้องใช้หน่วย sp

Posted by GoonerDev on July 23, 2015

เราเองเป็นนักพัฒนา android มาได้สักพัก พอมีฝีมืออยู่ท่าสองท่าเท่านั้น ยังไม่ใช่ผู้เชี่ยวชาญแบบหาตัวจับยากอะไร จุดเริ่มต้น เกิดขึ้นเนื่องจาก ได้ไปกินข้าวกับรุ่นพี่ๆที่สนิทกัน แลกเปลี่ยนความรู้กันถึงระดับ ผมก็โดนคำถามว่า

ทำไมขนาด font ของ android ต้องใช้หน่วย sp

เออ นั่นสิ เขียน android มาพักใหญ่ๆ ก็ไม่ได้เป็นประเด็นกับเรื่องนี้เหมือนกัน เมื่อมีความสงสัยเกิดขึ้น ก็จึงได้ฤกษ์หาข้อมูล

sp คืออะไร

sp ย่อมาจาก Scale-independent Pixels ซึ่งแปลได้ว่า ขนาด pixelที่เป็นอิสระ (google translate) ก็นะ…

จริงๆก็คือ การกำหนดขนาดโดยใช้ หน่วย sp นี้ ทำให้ขนาดเปลี่ยนแปลงได้ ก็คือ จะบอกว่า ไม่ได้ fixed ตายตัวนะจ๊ะ โดยมันสามารถขยายหรือย่อได้ ตามขนาดของหน้าจอ หรือตามแต่ที่ผู้ใช้งานกำหนดได้อีก

โดยขนาดที่เปลี่ยนไป เกิดขึ้นจากสูตร sp = px * ratio * scale โดยค่า ration จะไม่เปลี่ยน แต่ไอ้ตัว scale นี่แหละ ที่จะเปลี่ยนไปตามที่ผู้ใช้งานเครื่องกำหนด หรือตามขนาดหน้าจอ

ทำไมต้องใช้ sp

ก็เพราะว่ามันขยายได้ตามที่ผู้ใช้ต้องการน่ะสิ ให้คนตาไม่ค่อยดี มาอ่านตัวหนังสือเล็กๆก็คงไม่ไหวใช่มั๊ยล่ะ เมื่อขนาดเปลี่ยน อารมณ์ก็เปลี่ยน(อารมณ์ไรแว๊)

แล้วมันต่างจาก dp อย่างไรล่ะ

dp เกิดจากสูตร dp = px * ratio ซึ่งค่า ratio จะไม่เปลี่ยนเช่นเดียวกับ sp นั่นแหละ และมันยังไม่มีค่า scale มาเกี่ยวข้อง dp จึงเปลี่ยนตามความหนาแน่นของหน้าจออย่างเดียว ซึ่งถ้าอยากรู้เรื่องกำหนดขนาดด้วย dp แบบละเอียด ไปที่บล็อกนี้ครับ เขียนไว้ดีมาก

สรุป

ไม่มีอะไรมาก ที่ใช้หน่วย sp ก็เพราะมัน scale ได้นั่นแหละครับ เพราะฉะนั้นสหายทุกท่านก็อย่าลืมกำหนด layout ต่างๆให้รองรับการย่อขยายของ font ด้วย ไม่งั้นแอพเราก็ เอวัง
