Using facebook login Past1 (ติดตั้ง SDK)

Posted by GoonerDev on August 18, 2015

ลองเล่น Facebook SDK ตัวล่าสุด ซึ่งตอนนี้กลายเป็น version4 ไปเรียบร้อยโรงเรียนมาร์คซะแล้ว ก็เลยต้องมารื้อฟื้นปัดฝุ่นความรู้ตัวเราเองซะหน่อย

จะช้าอยู่ใย เข้าไปที่ developers.facebook.com แล้วก็ลองสร้างแอพใหม่ซะหน่อย ซึ่งก็แน่นอน android แอพพะย่ะค่ะ ใส่ชื่อแอพ เลือกหมวดหมู่ แล้วก็กด Create App ID เบาๆไปหนึ่งที

มาถึงหน้า download sdk แหล่มเลย มีstepมาให้เราด้วย ว่าต้องทำอะไรยังไง… ครับ จะรอใครมาตัดเค้กเปิดงานหรือไง เริ่มเลยสิครับ

Download the Facebook SDK for Android

ขั้นแรก facebook แนะนำให้ download facebook sdk มาก่อน แต่ว่าเราใช้ andrid studio ก็เลยไปโหลดจาก dependency ดีกว่า ชีวิตดี

Install Facebook on Your Emulator

การที่เราจะใช้ facebook login ได้ เราก็ต้องมีแอพ facebook ในเครื่องก่อนใช่มั๊ยล่ะ ถ้าเราใช้ real device ก็คงจะมีติดกันไว้แล้วละ ใครยังไม่มีก็ไปโหลดมาซะ ส่วนถ้าใครใช้ emultor ต่างๆก็ต้องโหลดตัว APK มาติดตั้งก่อนนะ ซึ่ง facebook ก็มีให้โหลดพร้อมวิธีติดตั้งในหน้านั้นเลย

Add Facebook SDK to Your Project

หลังจากมีแอพ facebook กันแล้ว เราจะเพิ่ม Facebook SDK ลงในโปรเจคเรา ผ่าน dependency ถ้าเรายังไม่มีโปรเจคก็ทำการสร้างขึ้นมาก่อนนะ โดย min sdk คือ 9(android 2.3) เมื่อมีโปรเจคแล้ว ก็เปิดไฟล์ build.gradle ของ app มาบัดเดี๋ยวนี้

เพิ่มคำสั่ง repositories { mavenCentral() } ก่อนคำสั่ง dependencies นะครับ เพื่อบอกว่าเราจะดึง library มาจาก maven central นั่นเอง หากสนใจเรื่องการ add library ต่างศึกษาต่อได้จาก บล็อกนี้ ซึ่งเป็นของคุณเอกครับ

จากนั้นก็เพิ่มคำสั่ง compile 'com.facebook.android:facebook-android-sdk:4.0.0' ลงใน dependencies แล้วก็ทำการ Sync gradle ซะ

Tell us about your Android project

กลับมาที่ facebook quick start จะให้ใส่ package name และ default activity class name ซึ่ง default activity class name ถ้าดูใน AndroidManifest.xml ก็คือ Activity ที่เป็นตัว android.intent.action.MAIN นั่นละจ้ะ

Add your development and release key hashes

ต่อมา facebook จะถามถึงตัว key hashes ของเรา เพื่อใช้สำหรับการ authen ระหว่างแอพเรากับ facebook

เราจะสร้าง Key Hashes บน command ครับ โดยรันคำสั่งดังนี้

Mac

keytool -exportcert -alias androiddebugkey -keystore ~/.android/debug.keystore | openssl sha1 -binary | openssl base64

Windows

keytool -exportcert -alias androiddebugkey -keystore %HOMEPATH%\.android\debug.keystore | openssl sha1 -binary | openssl base64

โดยตรง path สำหรับ keytore ตอนนี้เป็นตัว debug นะ ใช้สำหรับทดสอบได้ แต่ถ้าเราอัพขึ้น production หรือ playstore จะใช้ไม่ได้นะ ก่อนอัพก็อย่าลืม ไปเปลี่ยนก่อนล่ะ โดยใช้ keystore ตัวที่เราใช้อัพขึ้น production แล้วก็เอาไปใส่แทนใน setting ของแอพเราบน facebook

เอาละ ติดตั้ง Facebook SDK กันเรียบร้อย พร้อมใช้งานละ แต่รู้สึกว่าจะยาวละ ต่อที่บทความที่สองดีกว่า
